# Part-1 figure captions and data provenance

Reader-facing captions live in `../post.md` (the italic paragraph after each
image). This file keys every figure by its **reading-order number**, records the
provenance and computed values that were deliberately removed from the rendered
PNGs (maximal chrome-out: PNGs carry only the numbered title, axes, legend, and
data marks plus decode-required text), and pins the numbers.

Every number below is computed from repo artefacts by the figure scripts in this
directory (all import `_blogdata.py`). Realised returns and the price path come
from the leak-safe TSX data service
(`workshop_experiments.data_tsx.build_tsx_workshop_service`, Yahoo `^GSPTSE`
adjusted close). CRPS is recomputed per origin with
`properscoring.crps_ensemble` over the sorted quantile grid — the identical call
the scoring layer uses; it reproduces `data/results/*/leaderboard.csv`.

`_blogdata.landmarks()`; drawdowns are peak→trough, rebounds trough→recovery):

### Regenerating

From the repo root (data store co-located; no env var needed):

```
uv run python blog/part-1/assets/fig1_tsx_landmarks.py
uv run python blog/part-1/assets/fig2_crps_didactic.py
uv run python blog/part-1/assets/fig3_weekly_leaderboard.py
uv run python blog/part-1/assets/fig4_daily_crps_landmarks.py
uv run python blog/part-1/assets/fig5_quiet_vs_loud.py
```

Each writes its PNG beside the script through `_blogdata.savefig`, which
enforces the shared contract: 220 dpi, no figure-level text below the axes
block (footnotes are banned — they belong in the post caption and here), no
text under the 10 pt floor, and saved width ≤ 1.02 × figsize × 220 dpi. Scripts
print `CAPTION:` lines at save time with any computed values a caption needs —
paste from there, never hand-transcribe.
